# 系统管理 API

## 1. 健康检查

### 接口信息

- **路径**: `/api/health/`
- **方法**: `GET`
- **认证**: 不需要
- **描述**: 检查系统健康状态

### 请求示例

```bash
curl http://localhost:8000/api/health/
```

### 响应示例

```json
{
  "status": "ok",
  "database": "connected",
  "system_initialized": false
}
```

### 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| status | string | 系统状态，固定返回 "ok" |
| database | string | 数据库连接状态："connected" 或 "disconnected" |
| system_initialized | boolean | 系统是否已初始化 |
